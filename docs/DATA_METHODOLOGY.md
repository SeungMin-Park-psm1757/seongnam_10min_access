# 성남 10분 생활필수 접근권 격차 지도 데이터 방법론

## 1. 문제 정의

생활서비스가 행정구역 안에 존재하더라도 주민이 일상적으로 10분 안에 닿기 어렵다면 생활서비스 접근권은 충분하지 않다. 본 대시보드는 성남시 동별 생활필수 서비스 접근성을 비교하고, 지원수요가 큰 지역부터 정책 보강 우선순위를 제안한다.

## 2. 분석 단위와 입력 데이터

- 분석 단위: 성남시 동 단위 생활권
- 동별 지표: 인구, 고령비율, 1인가구 비율, 돌봄수요, 동 중심 좌표
- 서비스 위치: 의료기관, 약국, 버스 정류장, 돌봄센터 좌표
- 교체 방식: `data/accessibility_metrics.csv`, `data/service_points.csv`를 동일 컬럼으로 교체하면 실제 공공데이터 기반 분석으로 확장 가능

## 3. 점수 산정 구조

서비스별 접근 점수는 0~100점으로 표준화한다. 100점에 가까울수록 10분 생활권 안에서 접근성이 양호하다는 뜻이다.

- 의료 점수: 동 중심에서 의원·병원까지 10분 내 도달 가능성
- 약국 점수: 약국 접근성과 의료 거점과의 연계 가능성
- 교통 점수: 버스 정류장·환승 지점까지의 보행 접근성
- 돌봄 점수: 돌봄센터·복지서비스 거점까지의 생활권 접근성

종합 접근권 점수는 다음 가중치로 산정한다.

- 의료 30%
- 약국 20%
- 교통 25%
- 돌봄 25%

지원수요 지수는 고령비율, 1인가구 비율, 돌봄수요 등 취약성 관련 지표를 0~100점으로 표준화해 구성한다.

## 4. 정책 우선순위 산식

우선순위 점수는 접근성 부족과 지원수요를 함께 반영한다.

```text
우선순위 = (100 - 종합 접근권 점수) × 0.58 + 지원수요 지수 × 0.42
```

이 방식은 단순히 생활서비스가 적은 지역이 아니라, 접근성이 낮고 정책 필요도가 큰 지역을 먼저 드러내기 위한 것이다.

## 5. 정책 유형 분류

- 의료 보완형: 의료 또는 약국 점수가 가장 낮은 동
- 교통 보완형: 교통 점수가 가장 낮은 동
- 돌봄 보완형: 돌봄 점수가 가장 낮은 동
- 종합 취약형: 종합 접근권이 낮고 지원수요가 높은 동

각 유형은 “지금 할 일”, “중기 과제”, “협업 부서” 카드로 연결해 정책 실행 가능성을 높였다.

## 6. 시각화 원칙

- 점수 구간: 0-57 우선 보강, 58-69 주의, 70점 이상 양호
- 지도 표현: 원이 클수록 보강 필요도가 높고, 색이 붉을수록 접근권 점수가 낮다
- 산점도: 가로축은 10분 접근 어려움, 세로축은 지원수요
- 결측·추정치: 실제 데이터 적용 시 점선 배지 또는 별도 마커로 구분하도록 구조를 유지
