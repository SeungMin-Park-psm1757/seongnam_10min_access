# 성남 10분 생활필수 접근권 격차 지도 데이터 방법론

## 1. 문제 정의

생활서비스가 행정구역 안에 존재하더라도 주민이 일상적으로 닿기 어렵다면 체감 접근권은 충분하지 않다. 본 대시보드는 성남시 대표 생활권 10개 동의 의료, 약국, 교통 기반시설, 돌봄 거점 접근성을 비교하고 지원수요가 큰 지역부터 정책 보강 우선순위를 제안한다.

## 2. 분석 단위와 입력 데이터

- 분석 단위: 대표 생활권 10개 동
- 인구·수요: 성남시 행정동별 인구·세대 현황, 1인세대 현황
- 의료·약국: 성남시 의료기관 현황, 약국 현황
- 돌봄: 성남시 노인종합복지관 현황과 성남시 복지시설 좌표 PDF
- 교통: 경기교통정보센터 성남시 정류소 집계, OpenStreetMap 버스정류장 좌표
- 산출 파일: `data/accessibility_metrics.csv`, `data/service_points.csv`

## 3. 점수 산정 구조

서비스별 접근 점수는 0~100점으로 표준화한다. 100점에 가까울수록 대표 생활권 안에서 상대적으로 접근성이 양호하다는 뜻이다.

- 의료 점수: 대표 동에 매핑된 의료기관 수를 표준화
- 약국 점수: 대표 동에 매핑된 약국 수를 표준화
- 교통 점수: 가장 가까운 실제 버스정류장까지의 직선거리 기반 점수
- 돌봄 점수: 가장 가까운 노인종합복지관까지의 직선거리 기반 점수

종합 접근권 점수는 다음 가중치로 산정한다.

- 의료 30%
- 약국 20%
- 교통 25%
- 돌봄 25%

지원수요 지수는 고령층 비율, 1인세대 비율, 고령 1인세대 비율을 0~100점으로 표준화해 구성한다.

## 4. 정책 우선순위 산식

우선순위 점수는 접근성 부족과 지원수요를 함께 반영한다.

```text
우선순위 = (100 - 종합 접근권 점수) × 0.58 + 지원수요 지수 × 0.42
```

이 방식은 단순히 생활서비스가 적은 지역이 아니라, 접근성이 낮고 정책 필요도가 큰 지역을 먼저 드러내기 위한 것이다.

## 5. 정책 유형 분류

- 의료 보완형: 의료 또는 약국 점수가 가장 낮은 동
- 이동 접근 보완형: 교통 대리 점수가 가장 낮은 동
- 돌봄 보완형: 돌봄 점수가 가장 낮은 동
- 종합 취약형: 종합 접근권이 낮고 지원수요가 높은 동

각 유형은 “지금 할 일”, “중기 과제”, “협업 부서” 카드로 연결해 정책 실행 가능성을 높였다.

## 6. 한계와 확장

현재 교통 지표는 실제 버스정류장 좌표와의 직선거리 기반이다. 따라서 “실제 보행 10분 안 정류장 도달성”으로 단정하지 않는다. 보행 네트워크, 경사·횡단 여건, 민간 통신·카드 집계 데이터가 확보되면 같은 CSV 구조로 이동시간 기반 지표를 재산출할 수 있다.
