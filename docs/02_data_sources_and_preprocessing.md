# 02. 데이터 출처와 전처리

- 분석 단위: 대표 생활권 10개 동
- 선정 기준: 고령층 비중, 1인가구 비중, 생활서비스 접근 편차, 수정·중원·분당 권역 대표성
- 입력 데이터: `data/accessibility_metrics.csv`, `data/service_points.csv`
- 공공데이터 후보: 성남시 또는 공공데이터포털 의료기관 현황, 약국 현황, 버스정류장 위치, 복지·돌봄 생활서비스 거점 현황
- 인구·수요 데이터 후보: 행정동별 총인구, 고령층, 1인가구, 돌봄수요 관련 통계
- 민간데이터 확장: 공모전 제공 통신·카드 집계 데이터가 추가되면 수요·체류·이용 강도 보정 지표로 재산출 가능
- 전처리 원칙: 좌표 정제, 행정동 매핑, 서비스 유형 분류, 0~100 표준화, 결측치 확인 표시
- 갱신 구조: CSV 교체 방식으로 데이터 갱신 가능하며 행정동·격자·권역 단위 확장 가능
- 운영 활용: 정책 담당자가 최신 공공데이터로 주기적으로 교체해 같은 화면과 산식으로 재검토할 수 있다.
