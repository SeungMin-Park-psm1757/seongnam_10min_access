# 02. 데이터 출처와 전처리

- 분석 단위: 대표 생활권 10개 동
- 입력 데이터: `data/accessibility_metrics.csv`, `data/service_points.csv`
- 재산출 스크립트: `scripts/prepare-official-data.py`
- 실제 반영 데이터: 성남시 의료기관 현황, 약국 현황, 노인종합복지관 현황, 버스공영차고지 현황, 행정동별 인구·세대 현황, 1인세대 현황
- 좌표 보정: 의료·약국은 주소 지오코딩, 노인복지관은 성남시 복지시설 좌표 PDF를 함께 사용
- 교통 한계: 현재 교통 점수는 버스정류장 전수 접근성이 아니라 버스공영차고지 거리 기반 대리 지표
- 민간데이터 확장: 공모전 제공 통신·카드 집계 데이터가 추가되면 수요·체류·이용 강도 보정 지표로 재산출 가능
- 전처리 원칙: 좌표 정제, 대표 동 매핑, 서비스 유형 분류, 0~100 표준화, 한계 표시
- 갱신 구조: 원천 CSV를 갱신한 뒤 같은 스크립트를 실행하면 지도, 차트, 정책 카드가 함께 갱신된다.
