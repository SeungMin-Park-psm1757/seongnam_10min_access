# 제출용 대시보드 2차 정교화 결과

## 작업 범위

- CSV 원자료, 점수 산식, 서비스별 평균, 우선 확인 생활권, 정책 보완 유형 분류 로직은 변경하지 않았다.
- 수정 범위는 화면 밀도, 문구 길이, 여백, 카드 구조, 방법론 모달 구성, 발표용 캡처 품질이다.
- 유지한 대표 수치: 종합 접근성 70점, 의료 67점, 약국 71점, 교통 73점, 돌봄 71점.
- 유지한 태평1동 수치: 종합 접근성 69점, 접근 어려움 31점, 지원수요 96점, 고령층 27, 1인가구 38, 돌봄수요 96.

## 화면별 변경

| 화면 | 변경 전 | 변경 후 |
| --- | --- | --- |
| 1. 한눈에 보기 | KPI, 분석 범위, 산정 방식, 민감도, 데이터 출처가 한 화면에 함께 배치됨 | 제목, 핵심 해석, 지도, 70점 KPI, 우선 확인 생활권 TOP 3만 남김 |
| 2. 무엇이 부족한가 | 서비스 카드 안에 순위와 설명이 촘촘하게 들어감 | 평균 점수, 우선 확인 생활권 TOP 3, 정책 힌트만 강조 |
| 3. 어디가 우선인가 | 축 설명과 프로필 카드가 분석 문서처럼 보일 수 있음 | 축 문구를 짧게 하고 태평1동 프로필의 점수와 해석을 분리 |
| 4. 어떻게 보완할까 | 정책 카드가 표처럼 빽빽해질 위험이 큼 | 2x2 정책 브리프 카드로 정리하고 단기 점검/중기 검토 중심으로 축약 |
| 방법론 | 첫 화면의 보조 카드로 노출됨 | 상단 방법론 모달로 이동해 데이터 출처, 산식, 가중치, 민감도, 비식별 원칙을 확인 |
