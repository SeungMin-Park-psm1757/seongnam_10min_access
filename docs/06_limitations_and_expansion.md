# 06. 한계와 확장

- 현재 범위: 성남 전체 행정동 전수 분석 전 단계의 대표 생활권 10개 동 시각화
- 해석 방식: 특정 지역을 낙인찍는 순위표가 아니라 정책 점검 우선순위를 찾는 탐색 자료
- 산정 한계: 현재는 실제 보행 네트워크 이동시간이 아니라 위치 기반 근사 접근성 지표
- 보완 필요: 실제 정책 적용 전 현장 확인, 최신 데이터 검증, 주민 의견수렴 필요
- 데이터 확장: CSV 교체 방식으로 행정동·격자·권역 단위 데이터 확장 가능
- 민간데이터 확장: 통신 유동·체류, 카드 이용 데이터 제공 시 수요·이용 강도 지표 재산출 가능
- 고도화 방향: 보행 네트워크, 경사, 횡단보도, 운영시간, 배차간격, 야간·휴일 이용 가능성 반영
- 지속 활용: 담당 부서가 정기적으로 데이터를 교체하면 같은 산식으로 생활서비스 접근권 변화를 추적할 수 있다.
