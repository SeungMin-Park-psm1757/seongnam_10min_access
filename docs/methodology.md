# 성남 10분 생활필수 접근권 격차 지도 방법론

## 1. 분석 목적

이 대시보드는 성남시 생활필수 서비스가 단순히 행정구역 안에 있는지를 넘어, 시민이 실제로 이용하기 쉬운지를 비교하기 위한 정책 탐색 도구다. 의료·약국·교통·돌봄 접근성을 함께 보아 생활서비스 보강이 필요한 생활권을 찾는다.

## 2. 분석 단위: 대표 생활권 10개 동

현재 제출본은 성남시 전체 행정동 전수 분석 전 단계의 `대표 생활권 10개 동` 시각화다. 약점으로서의 표본이 아니라, 수정·중원·분당의 생활권 차이를 한 화면에서 설명하기 위한 대표 시나리오로 구성했다.

대표 생활권 선정 기준은 다음 네 가지다.

- 고령층 비중: 생활필수 서비스 접근성이 건강·돌봄 안전망과 연결되는 지역을 포함한다.
- 1인가구 비중: 정보 접근, 이동 지원, 생활서비스 의존도가 커질 수 있는 지역을 포함한다.
- 생활서비스 접근 편차: 의료·약국·교통·돌봄 점수 차이가 큰 지역을 포함한다.
- 권역 대표성: 수정구, 중원구, 분당구의 구도심·주거밀집·신도시 생활권이 함께 보이도록 구성한다.

전체 행정동 데이터가 확보되면 `data/accessibility_metrics.csv`와 `data/service_points.csv`를 같은 컬럼 구조로 교체해 전수 분석으로 확장할 수 있다.

## 3. 10분 접근권 산정 방식

현재 버전은 실제 보행 네트워크 기반 이동시간이 아니라 `위치 기반 근사 접근성 점수`다. 동 중심 좌표와 서비스 거점 위치, 서비스별 밀도·연결성을 바탕으로 접근성을 0~100점으로 표준화했다.

해석 원칙은 다음과 같다.

- 100점에 가까울수록 10분 생활권에 가까운 접근성을 가진다.
- 점수는 지역 간 상대 비교를 위한 지표이며, 특정 동의 좋고 나쁨을 단정하지 않는다.
- 실제 사업 적용 전에는 현장 조사, 보행 네트워크, 운영시간, 주민 의견수렴을 추가해야 한다.

향후 고도화에서는 보행 네트워크, 경사, 횡단보도, 보도 단절, 야간 운영 여부를 결합해 실제 10분 도달성으로 확장할 수 있다.

## 4. 핵심 산식

서비스별 접근성 점수는 의료, 약국, 교통, 돌봄 각각을 0~100점으로 표준화한다.

종합 접근권 점수는 다음 가중합으로 계산한다.

```text
종합 접근권 점수
= 의료 접근성 30%
+ 약국 접근성 20%
+ 교통 접근성 25%
+ 돌봄 접근성 25%
```

의료와 교통은 긴급성과 이동 기반성이 높아 상대적으로 큰 비중을 부여했다. 약국과 돌봄은 일상서비스 접근성과 생활지원 축으로 반영했다.

지원수요 점수는 다음 지표를 0~100점으로 표준화해 결합한다.

```text
지원수요 점수
= 고령층 비중 표준화
+ 1인가구 비중 표준화
+ 돌봄수요 지표 표준화
```

산점도의 X축은 다음과 같이 계산한다.

```text
10분 접근 어려움 = 100 - 종합 접근권 점수
```

## 5. 점수 구간과 임계값

현재 구간은 다음과 같다.

- 0~57점: 우선 점검 필요
- 58~69점: 주의 관찰
- 70점 이상: 대체로 양호

57점과 69점은 대표 생활권 10개 동의 점수 분포를 기준으로 하위권·중간권·상위권을 나누는 상대 기준이다. 현 제출본에서는 자연절단법 후보를 검토해 점수 간 간격이 커지는 지점을 경계로 삼고, 발표 화면에서 이해하기 쉬운 정수 구간으로 반올림했다.

전체 행정동 전수 분석으로 확장하면 동일 방식으로 경계값을 재산정하거나, 분위수·평균±표준편차 기준으로 민감도 비교를 추가할 수 있다.

## 6. 결측치와 확인 필요 데이터

결측치가 있는 경우 다음 원칙을 적용한다.

- 좌표 결측: 지도 표시에서 제외하고 원천 데이터 확인 대상으로 분류한다.
- 서비스 점수 결측: 동일 권역 평균 또는 인접 생활권 정보를 활용해 임시 산정하되, 결과 해석에는 `확인 필요`로 표시한다.
- 원천 데이터 기준시점 차이: 최신 데이터 기준으로 재계산하는 것을 원칙으로 한다.

## 7. 정책 해석

대시보드는 순위를 정하는 데서 멈추지 않고, 생활권을 의료 접근 보완형, 이동 접근 보완형, 돌봄 접근 보완형, 복합 보완 필요형으로 나눈다. 단독 유형이 보이지 않는 경우에도 빈칸으로 처리하지 않고, 대표 10개 동에서는 복합 보완 유형 안에서 함께 나타난다는 해석을 제공한다.
