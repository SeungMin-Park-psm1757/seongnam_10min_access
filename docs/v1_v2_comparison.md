# v1 / v2 비교

## 접근 경로

- `/v1`: 기존 버전. 기능 검증용 화면으로 보존.
- `/v2`: 디자인 개선 버전. 공모전 제출용 화면.
- `/`: v2와 동일한 제출용 기본 화면.

## v1

- 정보 구조와 기능 흐름을 검증하기에 적합하다.
- 지도, 서비스별 비교, 우선순위 산점도, 정책 보완 카드가 모두 한 흐름 안에 들어 있다.
- 다만 카드 내부 여백과 줄간격이 촘촘하고, 일부 라벨과 배지 반복이 생성형 AI 템플릿처럼 보일 수 있다.

## v2

- 기능과 데이터 구조는 유지한다.
- 타이포그래피, 여백, 카드 위계, 지도 명도를 조정했다.
- 첫 화면은 메시지와 대표 지표, 두 번째 화면은 서비스별 차이, 세 번째 화면은 우선순위, 네 번째 화면은 정책 보완 방향으로 역할을 분리했다.
- 방법론은 기존 모달 진입점을 유지하되 버튼과 모달 톤을 차분하게 정리했다.

## 검수 포인트

- 제목은 2줄 이내로 자연스럽게 보이도록 조정했다.
- 본문 line-height는 v2 스코프에서 넓게 적용했다.
- KPI 숫자와 설명 사이 간격을 분리했다.
- 지도와 우측 패널의 간격을 넓혔다.
- 정책 카드의 항목 간격을 넓히고 구분선 의존도를 줄였다.
- v1은 Git 태그 `v1-current`와 `/v1` 라우트로 함께 보존했다.
