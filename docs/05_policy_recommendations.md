# 05. 정책 제언

- 의료 접근 보완형: 동네의원, 약국, 야간·휴일 진료 정보를 한눈에 볼 수 있게 정리
- 의료 접근 보완형 협업: 보건소, 행정복지센터, 공공의료 협력기관
- 이동 접근 보완형: 정류장까지의 보행동선, 경사, 횡단 안전, 안내표지 우선 점검
- 이동 접근 보완형 협업: 교통기획과, 도로관리부서, 동 행정복지센터
- 돌봄 접근 보완형: 돌봄센터 이용 정보와 복지상담 창구를 통합 안내
- 돌봄 접근 보완형 협업: 복지정책과, 노인복지부서, 동 행정복지센터
- 복합 보완 필요형: 보건·교통·돌봄 상담을 동 단위 패키지로 안내
- 중기 과제: 생활SOC 거점, 이동지원, 방문서비스, 보행환경 개선을 묶어 검토
- 운영 방식: 정책 담당자가 최신 CSV로 지표를 주기적으로 갱신해 우선 점검 대상을 재확인
