# 지표 산식 요약

## 서비스별 접근성 점수

각 서비스 점수는 0~100점 상대 지표로 표준화한다.

```text
의료 접근성 = 의료기관 거리·밀도 기반 접근성 표준화 점수
약국 접근성 = 약국 거리·의료 거점 연결성 기반 접근성 표준화 점수
교통 접근성 = 정류장·환승 지점 거리와 생활서비스 이동 편의 표준화 점수
돌봄 접근성 = 돌봄·복지 거점 위치와 돌봄 수요 관계 표준화 점수
```

## 종합 접근권 점수

```text
종합 접근권 점수
= 의료 접근성 × 0.30
+ 약국 접근성 × 0.20
+ 교통 접근성 × 0.25
+ 돌봄 접근성 × 0.25
```

가중치 근거는 다음과 같다.

- 의료: 건강 안전망과 긴급성이 높아 30% 반영
- 약국: 일상 의약품 접근성과 의료 거점 보완 기능으로 20% 반영
- 교통: 다른 서비스 이용을 가능하게 하는 이동 기반성이 커 25% 반영
- 돌봄: 고령·복지 수요와 직접 연결되어 25% 반영

## 지원수요 점수

```text
지원수요 점수
= 고령층 비중 표준화 점수
+ 1인가구 비중 표준화 점수
+ 돌봄수요 표준화 점수
```

지원수요는 접근성 점수와 결합해 정책 우선순위를 해석한다.

## 산점도 축

```text
X축: 10분 접근 어려움 = 100 - 종합 접근권 점수
Y축: 지원수요 = 고령층·1인가구·돌봄수요 결합 점수
원 크기: 인구 규모
색상: 정책 보완 유형
```

## 점수 구간

```text
0~57점: 우선 점검 필요
58~69점: 주의 관찰
70점 이상: 대체로 양호
```

이 구간은 대표 생활권 점수 분포의 자연절단 후보를 기준으로 하위권·중간권·상위권을 나누는 상대 기준이다.
